# SKILL: toro-ui-defect-hunter-SKILL v1.0

## PURPOSE
Auditar interfaces para detectar errores UI/UX visibles, inconsistencias funcionales y defectos de presentación que afecten comprensión, confianza o uso.

## INPUT
- Screenshot de UI
- HTML/CSS/JS/TSX
- JSON de datos
- Descripción funcional

## CHECKLIST

### A. VISUAL DEFECTS
- texto truncado
- números cortados
- overflow
- padding inconsistente
- desalineación
- contraste bajo

### B. LOGICAL UI
- ingresos vs egresos no coherentes
- balance inconsistente
- porcentajes incorrectos
- ranking visual incorrecto

### C. STATES
- falta empty/loading/error states

CRITICAL / HIGH / MEDIUM / LOW
